### 安装karma环境
- npm i -g karma-cli
### 在项目中安装karma包
- npm i --save-dev karma 
### 安装karma插件
- npm i --save-dev karma-jasmine karma-chrome-launcher 
### 在工程目录中运行karma init 来进行环境初始化，并按照指示一步步完成
